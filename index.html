HTML

  • The head element contains information about the webpage.
  • Adding metadata tells the browser and search engines important information
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • A class selector can customise the formatting of elements
  • The CSS parser really cares about formatting eg 50 px is wrong, 50px is right.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls down the latest main branch to the local store
  • git checkout main: switches Git Bash to the main branch
  • git push origin feature/starter-code: pushes local changes to the remote GitHub repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript variable names are case sensitive.
  • JavaScript is a dynamic language with dynamic, weakly typed variables.
  • Strict equality (===) considers operands of different types to be different; equality (==) attempts to convert and compare operands that are of different types.
